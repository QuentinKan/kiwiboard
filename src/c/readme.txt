1. Download and install android NDK: http://developer.android.com/tools/sdk/ndk/index.html
2. Download and install android SDK: http://developer.android.com/sdk/index.html
3. You will find "ndk-build"(ndk-build.cmd on Windows platform) in NDK folder; and adb.exe in SDK sub folder. 
4. In project path(parent folder of jni), run "ndk-build" you will get compiled result.
5. Run "go.bat" to put it into KIWIBOARD and execute the program.
